<h2>📖 Overview</h2>
<p>
  This repository serves as a comprehensive framework for research into <b>AI-native wireless communication systems</b>. As we transition toward 6G, the integration of artificial intelligence into the physical and network layers becomes essential for handling massive connectivity, ultra-low latency, and dynamic spectrum environments.
</p>
<p>
  The core focus of this repository is the development of intelligent signal processing algorithms, specifically targeting <b>Automatic Modulation Classification (AMC)</b> and autonomous network management.
</p>

<hr>

<h2>📁 Current Projects</h2>

<h3>1. AI-Based Automatic Modulation Classification (AMC)</h3>
<p>
  Located in the <code>AI-Based-Automatic-Modulation-Classification</code> directory, this project implements a hybrid <b>ResNet-LSTM</b> architecture designed to:
</p>
<ul>
  <li>Extract spatial features from I/Q signal components using Deep Residual Learning.</li>
  <li>Capture temporal dependencies in signal sequences using Long Short-Term Memory (LSTM) layers.</li>
  <li>Achieve high classification accuracy across varying SNR levels on the RML2016.10b dataset.</li>
</ul>

<hr>

<h2>🔬 Research Vision</h2>
<p>The systems developed here are aligned with the following research objectives:</p>
<ul>
  <li><b>Intelligent Radio Access Networks (AI-RAN):</b> Implementing AI at the edge for real-time optimization.</li>
  <li><b>Agentic AI for 6G:</b> Developing autonomous AI agents capable of multi-step decision-making for interference resolution and resource allocation.</li>
  <li><b>Cognitive Radio:</b> Enhancing spectrum intelligence and dynamic spectrum access.</li>
</ul>
